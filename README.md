# 육회바른연어 통합 리뷰 대시보드

249개 매장의 **네이버지도 / 카카오맵 / 구글맵 / 배민 / 쿠팡이츠** 리뷰를 한 화면에서 모니터링.

## 라이브
https://jung917.github.io/yukhoe-reviews/

## 주요 기능
- **점포별 카드 그리드** — 검색·필터·정렬 (워닝만/데이터있음/평점순)
- **워닝 시스템** — 최근 7일 부정 리뷰 폭증 매장 자동 감지 (🔴빨강 / 🟡노랑)
- **채널별 비교** — 네이버/카카오/구글/배민/쿠팡이츠 리뷰 수 + 감성 분포
- **매장 상세 모달** — 30일 일별 추이 + 채널별 평점/링크 + 최근 5건
- **다크/라이트 테마**

## 데이터 출처
- 매장 마스터: `yby.co.kr/branch/find.php` (249개 공식 매장)
- 매장명 통일: `육회바른연어 ___점` 형식 표준화

## 워닝 기준
| 등급 | 조건 |
|------|------|
| 🔴 빨강 | 최근 7일 부정 ≥3건 OR 최근 30일 부정률 ≥50% (윈도우 5건 이상) |
| 🟡 노랑 | 최근 7일 부정 ≥2건 OR 최근 30일 부정률 ≥30% (윈도우 3건 이상) |

## 데이터 갱신
1. 5채널 크롤러가 각자 `~/yookhoe-review/data/channels/*.csv`에 저장
2. `merge_yukhoe_all.py`가 통합 → `yookhoe_reviews_latest.csv`
3. `dashboard_yukhoe_build.py`가 `dashboard.json` + `index.html` 생성
4. 본 repo의 `data/dashboard.json` 만 갱신 push → GitHub Pages 자동 배포
